# To-Do List App

A simple To-Do List application built using Java in Android Studio `2023.1.1.28-windows.exe`. This app allows users to add and delete tasks, helping them stay organized.

## Features

- Add items to the to-do list.
- Delete items from the to-do list.
- User-friendly and intuitive interface.



## Getting Started

### Prerequisites
- **Android Studio Version**: `2023.1.1.28-windows.exe` (64-bit)
- **Java Development Kit (JDK)**: Version 8 or above
- **Gradle**: Integrated with Android Studio

### Installation Steps
1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/todo-list-app.git

2. **Open the project in Android Studio**:

- Launch Android Studio.
- Click on `File > Open`.
- Navigate to the cloned repository folder and open it.

3. **Build the project**:
- Click on `Build > Rebuild Project` to ensure all dependencies are set up.

4. **Run the app**:
- Connect an Android device or start an emulator.
- Click on the `Run` button or press `Shift + F10`.
